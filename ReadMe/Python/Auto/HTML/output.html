1.操作前准备

（1）患者头部垫薄枕，使口、咽、喉三轴线尽量呈一致走向。操作者站于患者头侧。 2分

2.气管内插管操作过程

（1）右手拇指和示指呈“剪刀式”交叉，拇指推开患者下磨牙，示指抵住上门齿，打开口腔。 2分

（2）左手持喉镜，自口右侧口角置入，将舌体挡向左侧，再将镜移至正中，见到悬雍垂。 2分

（3）沿舌背弧度将镜再稍向前置入咽部，见到会厌。 2分

（4）提拉喉镜显露声门，右手以握笔状持导管从右侧弧形斜插口中，将导管前端对准声门后轻柔地插入气管内。 2分

（5）嘱助手拔出导管管芯，继续将导管送入（成人约2～3cm），导管尖端距离门齿约22±2cm。 2分

（6）放置牙垫，退出喉镜。 1分

（7）套囊充气，连接简易呼吸器。 1分

（8）初步确认气管导管位置：按压呼吸球囊人工通气，双侧胸廓对称起伏，听诊器双肺呼吸音固定存在且对称，初步确认气管导管位置正确。 2分

（9）固定导管，胶布将牙垫与气管导管固定于面颊，将头部复位。 1分

（10）连接呼吸机进行人工通气。 1分

3.职业素养：操作过程无菌观念强、动作规范、爱伤意识、良好的职业素养等。 2分
